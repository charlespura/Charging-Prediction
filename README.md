A simple static web app that predicts charging time for any rechargeable device, such as a phone, laptop, tablet, or power bank.

## Features

- Add a real charging example, such as `10%` to `75%` in `60 mins`.
- Predict how long a device needs to reach a goal battery percent.
- Save device charging profiles in the browser.
- Edit, use, and delete saved devices.
- Save prediction history.
- Show a visual progress bar.
- Toggle light and dark theme.
- Use dark theme by default on first visit.

## How To Use

Open `index.html` in a browser.

1. Enter the device name.
2. Add a charging example.
3. Enter current battery and goal battery.
4. Click `Predict Charging Time`.
5. Click `Save Device` to reuse that device profile later.
